## vue.js 小项目-  音乐播放器

项目中用到的数据是通过node.js爬取酷我官网数据
现在已经实现歌曲的播放，上一首下一首切换，以及列表循环，随机播放，单曲循环播放模式，喜欢收藏功能；
过渡动画transition的实现；
mv模块运用了video-player组件
